# Weather-app

This is an interactive *Streamlit web app* designed for analyzing, visualizing, and predicting weather data using a rule-based system. The app allows users to upload .xlsx files for various weather parameters, view merged data, analyze patterns with charts, and make basic weather predictions.

## 🚀 Features

- *Upload Excel files* for:
  - Humidity
  - Maximum Temperature
  - Minimum Temperature
  - Rainfall
  - Sunshine
  - Cloud Coverage
  - Wind Speed
- *Automatic processing* and merging of weather datasets
- *Visual analysis* with Altair:
  - Rainfall and wind speed category distribution
- *Rule-based prediction engine* for:
  - Rainfall (mm)
  - Wind Speed (m/s)

## 🧱 File Structure

- weather_app.py – The main Streamlit app file.
- requirements.txt – List of required Python packages.
